## Para Ejecutar

### Deploy servicio Graphcool
Entrar a la carpeta server y ejecutar:
yarn install o npm install
graphcool deploy

### Instalar dependencias e iniciar aplicacion.
En la carpeta ROOT ejecutar:
yarn install o npm install
yarn start
